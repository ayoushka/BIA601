## نظرة عامة

يُعد نظام التوصيات في المتاجر الإلكترونية أداة أساسية لمساعدة المستخدمين على اكتشاف المنتجات المناسبة، لكن التوصيات العشوائية أو العامة غالباً لا تلبي احتياجات كل مستخدم على حدة. يهدف هذا المشروع إلى تحسين جودة التوصيات باستخدام **الخوارزمية الجينية (Genetic Algorithm)** المستوحاة من نظرية الانتقاء الطبيعي. يقوم النظام بتحليل سلوك المستخدم (مشاهدة، نقر، شراء) وتقييماته السابقة، ثم يطبق عمليات التطور (الاختيار، التقاطع، الطفرة) على مجموعات من قوائم التوصيات لتوليد أفضل 5 منتجات مقترحة لكل مستخدم.

تم بناء المشروع استناداً إلى مقال علمي حديث (Deshmukh et al., 2024) الذي يجمع بين أشجار القرار والخوارزمية الجينية لأنظمة التوصية في مواقع التسوق عبر الإنترنت.

---

## الميزات الرئيسية

- **خوارزمية جينية حقيقية** – تمثيل الكروموسومات كقوائم منتجات، مع دالة لياقة مرجحة حسب سلوك المستخدم.
- **ملف مستخدم ديناميكي** – استخراج الفئات المفضلة، النطاق السعري، والمنتجات المستبعدة لكل مستخدم.
- **قائمة مرشحين ذكية** – تقليص مساحة البحث قبل تطبيق الخوارزمية الجينية.
- **واجهة ويب تفاعلية** – تصميم جذاب بلغتي العربية والإنجليزية، مع عرض التوصيات بشكل فوري.
-  **API احترافي** – خلفية FastAPI مع توثيق تلقائي عبر Swagger.
- **جاهز للنشر** – يمكن استضافته مجاناً على Render (API) و Netlify (واجهة).

## آلية عمل الخوارزمية الجينية

| المرحلة | الشرح |
|---------|-------|
| **تمثيل الحل (Chromosome)** | قائمة تحتوي على 5 أرقام تعريفية للمنتجات (product_id). |
| **دالة اللياقة (Fitness)** | `اللياقة = (Purchased × 10) + (Clicked × 5) + (Viewed × 1) + Avg_Rating` |
| **الاختيار (Selection)** | اختيار أفضل حلين (الأعلى لياقة) ليكون آباءً. |
| **التقاطع (Crossover)** | نقطة قطع واحدة لدمج المنتجات من الأبوين وتوليد أطفال. |
| **الطفرة (Mutation)** | تغيير منتج عشوائي في القائمة بمنتج جديد من قائمة المرشحين (احتمال 10%). |
| **الأجيال (Generations)** | 15 جيلاً لتحسين الحلول وتقاربها نحو التوصيات المثلى. |

## 🚀 كيفية تشغيل المشروع محلياً (لأعضاء الفريق)

### المتطلبات المسبقة
- Python 3.10 أو أحدث
- pip (مدير حزم Python)
- Git
- متصفح حديث

### الخطوات

1. **استنساخ المستودع**
   نكتب الأمر التالي في CMD
   git clone https://github.com/your-username/BIA601_Project.git
   cd BIA601_Project
2. **اعداد البيئة الخلفية**
نكتب في CMD
cd backend
pip install -r requirements.txt
محتوى requirements.txt
fastapi==0.115.6
uvicorn==0.34.0
pandas==2.2.3
numpy==2.2.1
openpyxl==3.1.5
4. **إضافة ملفات البيانات**
ضع ملفات Excel الأربعة (users.xlsx, products.xlsx, ratings.xlsx, behavior.xlsx) داخل مجلد data/ الموجود في جذر المشروع
6. **تشغيل الخادم الخلفي**
في CMD نكتب الأمر التالي
uvicorn main:app --reload
سترى رسالة " تم تحميل البيانات" ثم "Uvicorn running on http://127.0.0.1:8000".
8. **فتح الواجهة الأمامية**
يمكنك فتح ملف frontend/index.html مباشرة في المتصفح (أو استخدام Live Server في VS Code).
10. **تجربة التوصيات**
أدخل user_id موجود في قاعدة البيانات (مثال: 1، 2، 3…) ثم اضغط "احصل على توصيات ذكية".
ستظهر التوصيات المحسنة مع الفئة والسعر لكل منتج.
 إذا ظهر خطأ CORS، تأكد من إضافة CORSMiddleware في main.py (موجود بالفعل في الكود).
**فيديو شرح المشروع (3 دقائق)***
رابط الفيديو على Google Drive – تم تفعيل "أي شخص لديه الرابط يمكنه المشاهدة".
0:00 – مقدمة ومشكلة التوصيات العشوائية.
0:30 – شرح الخوارزمية الجينية وتشبيهها بالانتقاء الطبيعي.
1:00 – كيف طبقناها على توصيات المتاجر الإلكترونية.
1:40 – عرض حي للتطبيق (إدخال user_id ومشاهدة التوصيات).
2:20 – مقارنة النتائج قبل التحسين وبعده.
2:45 – الخاتمة والروابط.
